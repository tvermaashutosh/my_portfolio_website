# My Portfolio Website

Personal portfolio site built with React and Vite, deployed on Firebase Hosting.

## Tech Stack

- React
- Vite
- Firebase Hosting

## Run Locally

```sh
npm install
npm run dev
```

## Deploy

```sh
npm run build
firebase deploy
```

## Updating Portfolio Data

All portfolio data lives in `public/data.json`. Edit that file to update experience, education, achievements, skills, projects, etc.
